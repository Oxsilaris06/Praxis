À propos de Praxis Strategica

Praxis est un outil de planification opérationnelle conçu pour la Gendarmerie. Son objectif est de simplifier et d'optimiser la création d'Ordres Initiaux (O.I.) et de Rapports de Retour d'Expérience (RETEX).

En centralisant les informations clés et en automatisant la mise en forme, Praxis vous permet de vous concentrer sur la stratégie et l'exécution de vos missions, en vous assurant que la documentation est complète et cohérente.

Fonctionnalités clés

  • Génération d'Ordres Initiaux : Un formulaire détaillé vous guide à travers les différentes sections d'un O.I. (Situation, Adversaire, Exécution, Articulation, etc.).
  • Configuration de l'Unité : Le panneau de configuration vous permet d'ajouter et de modifier les membres de votre unité, ainsi que leurs équipements, pour une utilisation rapide dans vos documents.
  • Analyse par IA (via l'API Gemini) : Une fois l'opération terminée, vous pouvez soumettre les liens de vos rapports RETEX pour obtenir une analyse synthétique des points forts, des points faibles et des axes d'amélioration.
  • Exportation PDF : Générez des documents PDF professionnels et prêts à l'emploi en un seul clic.
  • Thèmes personnalisables : Adaptez l'interface à votre goût grâce à un mode clair et un mode sombre.

Tutoriel

Pour découvrir comment utiliser l'outil, un tutoriel interactif est disponible sur la page principale de Praxis. Il vous guidera étape par étape dans la création de votre premier Ordre Initial.

Comment ça marche ?

L'application fonctionne entièrement dans votre navigateur. Vos données sont sauvegardées localement et peuvent être importées ou exportées via un fichier JSON.

Pour commencer, configurez votre unité et, si vous le souhaitez, votre clé API Gemini via le menu "Setup". Ensuite, remplissez simplement le formulaire pour créer votre Ordre Initial et exportez le document au format PDF.
